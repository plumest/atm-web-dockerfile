# ATM-WEB-DOCKERFILE

## Installation
```
$ git clone https://github.com/plumest/atm-web-dockerfile
$ cd atm-web-dockerfile
$ docker build -t atm-docker .
$ docker run -d -p 9000:8090 atm-docker
```

now you can see my result on http://localhost:9000/home
